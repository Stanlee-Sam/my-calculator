
# Simple Calculator App

This is a basic calculator built with **Python** and deployed using **Streamlit**.
It performs simple arithmetic operations and provides an easy-to-use web interface.

## Features

* Add, subtract, multiply, and divide numbers
* Simple and clean user interface
* Error handling for invalid operations (like dividing by zero)
* Runs directly in the browser using Streamlit

## How It Works

The app takes two numbers as input and lets you pick an operator from a dropdown menu.
When you click **Calculate**, it processes the operation and shows the result instantly.

## How to Run the App

1. **Clone the repository**

```
git clone https://github.com/your-username/your-repo-name.git
```

2. **Install dependencies**

```
pip install streamlit
```

3. **Run the app**

```
streamlit run app.py
```
